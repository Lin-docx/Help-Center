## 功能概述

通知中心是您获取 Zleap 动态的信息中枢，集中管理所有与您相关的系统消息、团队动态和产品变更。点击界面右上角的通知铃铛图标可随时访问。

## 访问入口

### 网页版

登录您的 Zleap 账号，点击界面右上角<strong style={{ fontWeight: 700 }}>「通知铃铛」</strong>图标，即可查看相关的<strong style={{ fontWeight: 700 }}>「通知」</strong>、<strong style={{ fontWeight: 700 }}>「活动」</strong>和<strong style={{ fontWeight: 700 }}>「更新日志」</strong>。

<img src="/images/通知中心按钮.gif" alt="通知中心按钮" style={{ maxWidth: '100%', height: 'auto' }} />

## 板块介绍

### 通知

这是您的个人消息收件箱，用于接收需要您关注或处理的即时信息，包括新报告生成/失败、订阅报告生成等系统提醒。点击相关通知，可查看详情并跳转至相关内容。

<img src="/images/通知.gif" alt="通知" style={{ maxWidth: '100%', height: 'auto' }} />

### 活动

Zleap 官方发布的最新的线上/线下活动、社区互动及重要计划等通知。

<img src="/images/活动.gif" alt="活动" style={{ maxWidth: '100%', height: 'auto' }} />

### 更新日志

Zleap产品功能的最新优化与升级，点击可查看详细的优化与升级信息。

<img src="/images/更新日志.gif" alt="更新日志" style={{ maxWidth: '100%', height: 'auto' }} />

通过合理利用通知中心，您可以在高效处理信息的同时，随时了解 Zleap 的官方动态和产品进化。
