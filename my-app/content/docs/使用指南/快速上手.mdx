## 第一步：创建信息源

<div style={{ textAlign: 'justify' }}>

信息源是生成报告的基础，Zleap 支持上传文档或添加链接作为信息源输入。

### 1.1 点击信息管理

登录 Zleap 后，点击首页中的<strong style={{ fontWeight: 700 }}>「信息管理」</strong>，进入信息源管理页面。

然后，点击左侧菜单中的<strong style={{ fontWeight: 700 }}>「创建信息源」</strong>，点击<strong style={{ fontWeight: 700 }}>「文档上传」</strong>，通过上传的文档作为信息源类型。

<img src="/images/点击信息管理.gif" alt="点击信息管理" style={{ maxWidth: '100%', height: 'auto' }} />

### 1.2 上传文档

点击<strong style={{ fontWeight: 700 }}>「文档上传」</strong>后，进入“上传信息源”界面。

- 点击<strong style={{ fontWeight: 700 }}>「选择文件」</strong>，上传本地文档。目前，Zleap 支持 pdf、txt、md、docx、pptx等格式，以及粘贴网页链接。
- 文件上传成功后，填写<strong style={{ fontWeight: 700 }}>「信息名称」</strong>；
- 填写完毕，点击右上方<strong style={{ fontWeight: 700 }}>「提交信息」</strong>，完成上传。

<img src="/images/上传信息源.gif" alt="上传信息源" style={{ maxWidth: '100%', height: 'auto' }} />

### 1.3 关联应用

信息提交成功后，系统将弹出关联应用页面，请点击<strong style={{ fontWeight: 700 }}>「关联应用」</strong>。

并在弹出页面中选择<strong style={{ fontWeight: 700 }}>「报告助手」</strong>。

<img src="/images/关联应用.gif" alt="关联应用" style={{ maxWidth: '100%', height: 'auto' }} />

至此，您的信息源已创建完成。

系统将自动跳转至报告生成页面。

</div>

## 第二步：生成 AI 报告

<div style={{ textAlign: 'justify' }}>

### 2.1 选择信息源

系统跳转到报告生成页面后，点击左侧<strong style={{ fontWeight: 700 }}>「创建报告任务」</strong>，在弹出页面中选择您刚创建的信息源。

选择后，点击页面空白处激活<strong style={{ fontWeight: 700 }}>「下一步」</strong>按钮，并点击进入。

<img src="/images/创建报告任务.gif" alt="创建报告任务" style={{ maxWidth: '100%', height: 'auto' }} />

随后，系统会弹出一个<strong style={{ fontWeight: 700 }}>「创建报告任务」</strong>页面。

### 2.2 创建报告任务

在“创建报告任务”页面中，您可根据需要：

- 选择想要的报告模板
- 输入相关的提示词
- 选择喜欢的报告风格

由于本次教程的信息源为上传的文档，所以执行方式选择<strong style={{ fontWeight: 700 }}>「单次立即执行」</strong>即可。

完成相关配置后，点击<strong style={{ fontWeight: 700 }}>「创建」</strong>，系统将开始自动生成报告。

<img src="/images/创建报告任务-选择.gif" alt="创建报告任务" style={{ maxWidth: '100%', height: 'auto' }} />

### 2.3 生成报告

系统将在短时间内完成报告生成。

报告生成后，您可在左侧导航栏的<strong style={{ fontWeight: 700 }}>「任务管理」</strong>中查看所有报告及状态。

<img src="/images/任务管理.png" alt="任务管理" width="400" style={{marginTop: '2px'}} />

点击对应的报告卡片，即可进入报告详情页，查阅由 Zleap 生成的完整 AI 智能报告。

在报告页面中，您还可通过右侧的AI 对话按钮，直接使用自然语言提问，AI 将基于报告内容进行精准解答。

<img src="/images/AI报告.jpg" alt="AI报告" width="400" style={{marginTop: '2px'}} />

至此，您已完成 Zleap 从账户创建到报告生成的全流程。

快来创建您的第一份报告，体验 AI 驱动的高效信息管理吧！

</div>
