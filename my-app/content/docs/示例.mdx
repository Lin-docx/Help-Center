## 录屏示例

经过多次测试，觉得全部录全屏比较好，可以让教程界面的长度宽度都统一。

### 方式一：直接录屏

快速，衔接性好，流畅自然，可以直接加边框不变形。

<img src="/images/示例-全屏.gif" alt="示例-全屏" style={{ maxWidth: '100%', height: 'auto' }} />

### 方式二：后期处理

需要先录视频，后期剪辑和制作，再制作成动图，优点是页面相对好一点，但耗费时间会更多些。

<img src="/images/示例-全屏2.gif" alt="示例-全屏2" style={{ maxWidth: '100%', height: 'auto' }} />

### 方式三：直接放视频

需要先录视频，然后剪辑，同样要耗费多点时间，但比方式二少几个步骤。

<video controls src="/video/视频3.mp4" style={{ maxWidth: '100%', height: 'auto' }} />

我个人更倾向于方式三，直接放全屏的，经过剪辑处理的视频。
