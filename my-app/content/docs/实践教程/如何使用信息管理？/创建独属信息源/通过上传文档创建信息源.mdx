---
title: 通过上传文档创建信息源

---

<div style={{ textAlign: 'justify' }}>

将本地文档上传至 Zleap，是构建您个人专属信息源的第一步。完成上传后，此信息源即可作为 AI 分析的来源，用于后续的智能报告生成、内容问答与深度搜索。

## 第一步：进入创建页面

- 登录 Zleap，点击首页的<strong style={{ fontWeight: 700 }}>「信息管理」</strong>，进入上传信息源页面。
- 点击<strong style={{ fontWeight: 700 }}>「文档上传」</strong>。

<img src="/images/创建信息源-文档上传.gif" alt="创建信息源-文档上传" style={{ maxWidth: '100%', height: 'auto' }} />

## 第二步：上传并设置文档

- 在“上传信息源”页面，点击<strong style={{ fontWeight: 700 }}>「选择文件」</strong>，上传您的文档。目前，Zleap 支持 pdf、txt、md、docx、pptx 等格式，以及粘贴网页链接。
- 文件上传成功后，在<strong style={{ fontWeight: 700 }}>「信息名称」</strong>字段中，输入清晰的名称。
- 确认信息无误后，点击右上角的<strong style={{ fontWeight: 700 }}>「提交信息」</strong>按钮。

<img src="/images/文档上传-选择文件.gif" alt="文档上传-选择文件" style={{ maxWidth: '100%', height: 'auto' }} />

## 第三步：完成创建

- 提交成功后，页面会提示“信息源创建成功”。
- 系统会弹出“关联应用”窗口。

  方式一：点击<strong style={{ fontWeight: 700 }}>「关联应用」</strong>，并选择<strong style={{ fontWeight: 700 }}>「报告助手」</strong>，为下一步生成报告做好准备。

<img src="/images/文档上传-关联应用.gif" alt="文档上传-关联应用" style={{ maxWidth: '100%', height: 'auto' }} />

  方式二：点击右上角的“<strong style={{ fontWeight: 700 }}>×</strong>”号，先行完成信息源的创建。


</div>
