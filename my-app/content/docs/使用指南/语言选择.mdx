## 功能概述

Zleap 提供完整的界面语言切换功能，旨在为全球用户与团队提供更友好、个性化的产品使用体验。目前，Zleap 支持简体中文与 English，您可以根据个人偏好与所在地区随时更改。

## 操作步骤

### 网页版切换语言

登录您的 Zleap 账号，在界面右上角找到<strong style={{ fontWeight: 700 }}>「语言切换」</strong>图标。

点击该图标，系统会弹出语言选择菜单，点击选择您希望使用的语言<strong style={{ fontWeight: 700 }}>「中文」</strong>或 <strong style={{ fontWeight: 700 }}>「English」</strong>。

<img src="/images/语言选择.gif" alt="语言选择" style={{ maxWidth: '100%', height: 'auto' }} />

点击后，页面将自动刷新，界面文字将更新为您选择的语言。

</div>

## 常见问答 (Q&A)

<div style={{ textAlign: 'justify' }}>

### 部分内容语言没变化

<strong style={{ fontWeight: 700 }}>Q：</strong>我切换了语言，但为什么部分内容语言没有变化？

<strong style={{ fontWeight: 700 }}>A：</strong>语言切换主要作用于 Zleap 产品本身的界面元素。以下内容可能不会改变：

- 您创建的、本身包含特定语言的信息源和报告
- 您的浏览器或操作系统设置可能影响字体显示，但不影响语言本身

### 可以自定义其他语言吗

<strong style={{ fontWeight: 700 }}>Q：</strong>除了中文与英文外，还可以选择其他语言吗？

<strong style={{ fontWeight: 700 }}>A：</strong>目前 Zleap 官方支持的语言为简体中文和English。更多语言支持已在我们的产品规划中，敬请关注官方公告。

</div>
