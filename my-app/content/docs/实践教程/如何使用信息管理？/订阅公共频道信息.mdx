## 功能概述

公共频道是 Zleap 内置的优质信息源集合，覆盖多平台、多领域与高质量创作者。该功能支持您以报告形式浏览全部内容，也可订阅感兴趣的领域或博主，实现对公开信息的集中管理。

借助公共频道，您可以从海量信息中精准筛选高价值内容，提升信息获取效率。

## 操作教程

### 1. 进入公共频道

点击首页的<strong style={{ fontWeight: 700 }}>「信息管理」</strong>，进入<strong style={{ fontWeight: 700 }}>「公共频道」</strong>界面。

<img src="/images/进入公共频道.gif" alt="进入公共频道" style={{ maxWidth: '100%', height: 'auto' }} />

当前，公共频道已汇聚来自 AI、时政、商业、职场等领域的数千条优质信息源，持续为您提供高质量的公开信息。

### 2. 订阅公共频道

#### 2.1 订阅频道

选择需订阅的频道，在对应频道卡片右侧点击<strong style={{ fontWeight: 700 }}>「订阅」</strong>按钮。

Zleap 支持订阅多个频道，您可任意订阅喜欢的频道。

<img src="/images/点击订阅.gif" alt="点击订阅" style={{ maxWidth: '100%', height: 'auto' }} />

若想取消订阅，在订阅成功后，点击卡片右侧<strong style={{ fontWeight: 700 }}>「已订阅」</strong>按钮，即可取消订阅。

#### 2.2 查看订阅内容

订阅后，在推荐版块可看到订阅频道。

您还可通过右上方<strong style={{ fontWeight: 700 }}>「我的订阅」</strong>入口，查看和管理所有已订阅频道。

<img src="/images/我的订阅.gif" alt="我的订阅" style={{ maxWidth: '100%', height: 'auto' }} />

点击任一频道，即可进入其最新发布的信息列表；点击具体的<strong style={{ fontWeight: 700 }}>「事件名称」</strong>，可跳转至原文详情页。

<img src="/images/我的订阅-原文详情.gif" alt="我的订阅-原文详情" style={{ maxWidth: '100%', height: 'auto' }} />

#### 2.3 使用 AI 助手

在频道信息首页，您还可使用<strong style={{ fontWeight: 700 }}>「搜索助手」</strong>进行AI驱动的信息检索，或使用<strong style={{ fontWeight: 700 }}>「问答助手」</strong>进行AI驱动的智能问答。

<img src="/images/我的订阅-AI助手.gif" alt="我的订阅-AI助手" style={{ maxWidth: '100%', height: 'auto' }} />

### 3. 浏览全部内容

#### 3.1 选择内容模式

点击右上角切换按钮，选择<strong style={{ fontWeight: 700 }}>「报告内容」</strong>模式。

在该模式下，信息将以卡片形式展示，包含标题与摘要，方便快速浏览。

<img src="/images/报告内容模式.gif" alt="报告内容模式" style={{ maxWidth: '100%', height: 'auto' }} />

#### 3.2 开始阅读信息

点击卡片，进入预览页后点击<strong style={{ fontWeight: 700 }}>「开始阅读」</strong>，即可浏览内容简介。

如需查看完整信息报告，点击<strong style={{ fontWeight: 700 }}>「详情」</strong>即可。

<img src="/images/报告内容-详情.gif" alt="报告内容-详情" style={{ maxWidth: '100%', height: 'auto' }} />

#### 3.3 个性化设置

在信息报告界面，您还可以进行以下操作：

- <strong style={{ fontWeight: 700 }}>切换阅读模式</strong>：点击右上角“卡片模式”图标，可切换不同阅读模式。

- <strong style={{ fontWeight: 700 }}>分享报告内容</strong>：点击右上方分享按钮，可将该报告一键分享至微信等外部平台。

- <strong style={{ fontWeight: 700 }}>使用 AI 对话</strong>：点击右侧“AI 对话”按钮，即可直接输入问题，AI 将基于当前报告内容提供精准解答。

<img src="/images/报告内容-个性化设置.gif" alt="报告内容-个性化设置" style={{ maxWidth: '100%', height: 'auto' }} />
